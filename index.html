Analista de Seguridad Fisica / Web Developer

Sobre mi

Me encuentro actualmente trabajando en la gerencia de Seguridad de Activos y Prevención de Fraudes del Banco de Córdoba.
Me considero una persona curiosa y dispuesta a nuevos desafíos. Siempre buscando innovar y crecer profesionalmente. Creo que todo se trata de procesos y que con constancia y disciplina todo se puede.

Mi experiencia

Analista de Seguridad
Banco de Córdoba

Me desempeño como analista en la Subgerencia de Seguridad para el Banco de Córdoba. Dentro de mis tareas y responsabilidades se encuentran:
- Administrar el sistema completo de control de acceso a todas las dependencias del Banco.
- Administrar los distintos sistemas de CCTV y alarmas bancarias.
- Gestionar y monitorear el proyecto de actualización del parque de CCTV.
- Analizar desviaciones del programa de seguridad de Banco de Córdoba.
- Delegar tareas a distintos proveedores.
- Encontrar áreas de mejora para la seguridad de Banco de Córdoba.
- Resolver consultas de seguridad para los clientes internos.
